Enumerating the Web Application

I first went to port 80 and was greeted with a very basic webpage. The webpage did contain links using the hostname spectra.htb; so I added that to my /etc/hosts file.

The link to the test url results in an error page claiming there was a database error. No I found no other interesting information here.

Enumerating Wordpress

The url http://spectra.htb/main/ links to a wordpress website. I tried logging in with admin:admin on http://spectra.htb/main/wp-login.php and got an error message that the username is unknown. I may thus leverage this to enumerate users.

Clicking on the comment field I discover that the wordpress version is 5.4.2
